# NMIT BIT Taster Session: Creative Coding with p5.js

Welcome to the Nelson Marlborough Institute of Technology (NMIT) **Bachelor of Information Technology (BIT)** high school taster session resource repository. 

This repository contains all the materials required to host a **1-hour-45-minute (105 minutes)** hands-on workshop introducing programming basics using **[p5.js](https://p5js.org/)**—a visual JavaScript library. The workshop is designed to show students that coding is highly visual, creative, and accessible, while showcasing the practical learning structure of the NMIT BIT degree.

---

## 🗂 Repository Structure

Here is an overview of the materials provided in this repository:

*   **[presentation.html](file:///c:/Users/alika/workspace/tasterSOne/presentation.html)**: A self-contained, responsive HTML slide deck designed in a premium, dark-mode glassmorphic theme. It features keyboard slide navigation (`Left` / `Right` arrows), custom syntax-highlighted code guides, and an **embedded live p5.js Web Editor** so students can live-code directly inside the slides.
*   **[lesson_plan.md](file:///c:/Users/alika/workspace/tasterSOne/lesson_plan.md)**: A detailed, minute-by-minute facilitator master guide. It covers talking points, guided coding steps, NMIT BIT program hooks, and a facilitator watchlist for common student coding errors.
*   **[student_cheatsheet.md](file:///c:/Users/alika/workspace/tasterSOne/student_cheatsheet.md)**: A print-friendly reference card outlining canvas coordinates, basic shape parameters, RGB colors, opacity blending, built-in variables, and a quick-start template code.
*   **`demos/`**: Progressive p5.js code scripts to help students build step-by-step:
    *   **[demo1_basics.js](file:///c:/Users/alika/workspace/tasterSOne/demos/demo1_basics.js)**: Introduction to canvas, coordinate axes, and drawing static shapes.
    *   **[demo2_interactive.js](file:///c:/Users/alika/workspace/tasterSOne/demos/demo2_interactive.js)**: Mouse-tracking neon brush tool displaying variables, random functions, opacity trails, and clearing with a click.
    *   **[demo3_game.js](file:///c:/Users/alika/workspace/tasterSOne/demos/demo3_game.js)**: A functional bubble catching game with gravity physics, collision boundaries, and live scorekeeping.

## 🚀 How to Run the Lesson

### For Facilitators / Teachers
1.  **Launch slides**: Simply open `presentation.html` in any web browser. Use **Arrow Right (▶)**, **Spacebar**, or **Enter** to advance slides, and **Arrow Left (◀)** to return.
2.  **Refer to guide**: Keep `lesson_plan.md` open on a separate screen or printed as your cheat sheet for key talking points.
3.  **Live Code**: Use Slide 8 (Live Code Playground) to demonstrate code changes on the projector directly within the presentation slide!

### For Students
1.  Navigate to the **[p5.js Web Editor](https://editor.p5js.org/)** to begin. No installation or registration required.
2.  Use the **[student_cheatsheet.md](file:///c:/Users/alika/workspace/tasterSOne/student_cheatsheet.md)** to copy sample codes and look up coordinate geometry commands.
