Guided Build: Neon Glow Brush

We will make a dynamic drawing canvas where circles track our mouse and shift colors.

function setup() {
  createCanvas(windowWidth, windowHeight);
  background(15, 23, 42); // Slate Blue (drawn once!)
}

function draw() {
  if (mouseIsPressed) {
    background(15, 23, 42); // Clear board if clicked
  }
  
  fill(random(100,255), random(150,255), random(200,255), 120);
  noStroke();
  ellipse(mouseX, mouseY, 40, 40);
}

💡 Pro Tip: Moving `background()` to setup lets our brush strokes persist and leave a beautiful neon paint trail!

Creative Play & Lab Time

Now it's your turn. Use the reference sheets and make this sketch your own!

  • Level 1: Change the shapes. Can you draw neon squares (`rect`) instead of circles? Can you make the brush size larger?
  • Level 2: Change colors. Alter the numbers in `random()` to make it purely green, hot pink, or purple.
  • Level 3: Try the **Bubble Catch Game** template (`demos/demo3_game.js`). Customize the speeds and scores!
